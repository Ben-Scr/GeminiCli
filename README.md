# GeminiCli

A C# command-line tool for interacting with ``Google Gemini models`` directly from the terminal.

## Setup

Follow these steps to get started after cloning the repository.

### 1. Get a Gemini API Key
Create an API key in **Google AI Studio**:

https://aistudio.google.com/api-keys

### 2. Configure Environment Variables
Open the `.env` file located at ``GeminiCli/.env`` and replace the placeholder ``API_KEY`` with your key
